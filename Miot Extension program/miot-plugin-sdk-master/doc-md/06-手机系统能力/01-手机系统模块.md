## miot/system
扩展程序运行时手机系统提供的功能
系统的能力主要包括：
电量(battery.js)
屏幕(screen.js)
音量(volume.js)
加速计(accelerometer.js)
罗盘(compass.js)
陀螺仪(gyroscope.js)
性能(memory.js)
扫码(scancode.js)
震动(vibrate.js)
语音、相机(permission.js)
定位(location.js)
网络(network.js)

**Export**: public  
**Doc_name**: 手机系统模块  
**Doc_index**: 1  
**Doc_directory**: system  
**Example**  
```js
import {System} from 'miot'
```
